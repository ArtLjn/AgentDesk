# AI Agent 开发工程师学习实战项目

本项目是AI Agent开发工程师学习路线的实战代码库，包含3个梯度级别的实战项目：

1. 基础Agent实现（ReAct/Plan-Execute/Reflexion）
2. RAG系统开发（个人知识库、论文阅读助手）
3. 企业级多Agent自动化系统

## 环境要求
- Python 3.10+
- OpenAI/DeepSeek API Key

## 快速开始
1. 克隆项目
2. 安装依赖：`pip install -r requirements.txt`
3. 复制.env.example为.env并配置API Key
4. 运行示例：`python examples/demo.py`
